# Data Structures Implementation (Stack)

## Project Description
This project demonstrates the implementation of the Stack data structure in C++ using basic stack operations. The program allows users to perform Push, Pop, Display, and Exit operations through a menu-driven interface.

## Features
- Push element into the stack
- Pop element from the stack
- Display all stack elements
- Menu-driven program
- Handles Stack Overflow and Stack Underflow conditions

## Files Included
- `stack.cpp` – C++ source code
- `README.md` – Project documentation

## How to Run
1. Open `stack.cpp` in Dev-C++.
2. Compile and Run the program.
3. Choose the desired operation from the menu.
4. Enter values as required.

This project demonstrates the implementation of a Stack using C++. It covers the fundamental stack operations such as Push, Pop, and Display while handling overflow and underflow conditions.
